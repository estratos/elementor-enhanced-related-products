# Elementor Enhanced Related Products

Advanced WooCommerce related products widget for Elementor with complete filtering control.

## Features

- 5 filtering logics: Category, Tag, Both (AND), Combined (OR), Manual
- Exact HTML structure matching Elementor's native Products widget
- Complete styling controls in Elementor editor
- Responsive grid system
- Compatible with all WooCommerce themes

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin in WordPress
3. Add widget to Elementor single product templates

## Usage

1. Edit a Single Product template in Elementor
2. Search for "Enhanced Related Products" widget
3. Add it to your template
4. Configure filtering logic and styling options

## Filtering Options

- **Category Only**: Show products from same categories
- **Tag Only**: Show products with same tags  
- **Both (AND)**: Products sharing both category AND tag (WooCommerce default)
- **Combined (OR)**: Products sharing category OR tag (more products)
- **Manual**: Select specific products manually
